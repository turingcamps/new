_Course Offerings_

Python Pros

A beginner course for Python, the most used coding language worldwide. Every concept is taught through hands-on projects, ending with a hackathon (a coding competition).

Game Dev

Don't just play games, make them! Students will build a video game, from start to finish using Unity, a professional-grade tool, while learning about game logic, physics, and user input.

Robotics

A highly interactive course where students write code that controls the physical world. Using Arduino boards, program sensors, lights, and motors to build hands-on projects.

Cybersecurity

Explore cybersecurity by learning about data, encryption, networks, and ethical hacking. Cybersecurity is a fast-growing field, and this course provides exposure to real-world skills.
